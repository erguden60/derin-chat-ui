Hızlı Başlangıç

Modern proje yapıları için NPM paketini, hızlı prototipler veya legacy sistemler için CDN sürümünü kullanabilirsiniz.

İpucu: Mock modu sayesinde backend ekibi API'yi hazırlarken siz UI entegrasyonunu bitirebilirsiniz.

main.js 
import DerinChat from 'derin-chat';

DerinChat.init({
  // Backend hazır değilse bunu true yapın
  mock: true,

  ui: {
    position: 'bottom-right',
    logo: 'https://site.com/logo.png',
    colors: {
      primary: '#6366f1'
    }
  },

  user: {
    id: 'u_123',
    name: 'Emirhan'
  }
});

API Konfigürasyonu

Aşağıdaki tüm parametreler DerinChat.init({}) fonksiyonu içinde kullanılabilir.

Parametre Tip Varsayılan Açıklama
mock boolean false ÖNERİLENTrue ise backend'e bağlanmaz, simülasyon yapar.
apiUrl string - Backend endpoint adresi. mock: false ise zorunludur.
apiKey string - Güvenlik anahtarı. Header'da Bearer token olarak gönderilir.
ui.colors object Indigo Renk paleti (primary, userMessageBg, botMessageBg vb.).
ui.position string 'bottom-right' Konum. 'bottom-right' veya 'bottom-left'.
ui.logo string - Header kısmında görünecek avatar veya logo URL'i.
ui.texts object - Metinler (title, placeholder, loading vb.).
ui.fontFamily string System Widget fontu. Örn: 'Inter, sans-serif'.
user object - Kullanıcı verisi (id, name, email). Backend'e iletilir.
behavior.openOnLoad boolean false Sayfa yüklendiğinde pencere otomatik açılır.
